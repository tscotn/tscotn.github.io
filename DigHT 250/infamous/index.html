Social media, especially Twitter, is full of bots. From influencing politics, monetizing trends, and creating art, bots have been used for ends both good and bad. This page is a resource for information about Twitter bots, and some examples of my own experience in building them.

What Are They

"A Twitter bot is a type of bot software that controls a Twitter account via the Twitter API. The bot software may autonomously perform actions such as tweeting, re-tweeting, liking, following, unfollowing, or direct messaging other accounts. The automation of Twitter accounts is governed by a set of automation rules that outline proper and improper uses of automation. Proper usage includes broadcasting helpful information, automatically generating interesting or creative content, and automatically replying to users via direct message. Improper usage includes circumventing API rate limits, violating user privacy, spamming, and sockpuppeting."

Wikipedia

Tools for Detecting Bots

When problematic bots that promote disinformation or otherwise make social media a more dangerous place, it's vital to be able to identify whether an account is a bot or not. These are a few tools that can be installed for determining the validity of specific accounts.

  • Botometer
  • Bot Sentinel
  • BotSight

How to Build a Bot

There are several ways to built and maintain a Twitter bot, but you'll always need to follow the same general steps:

  1. Apply for a Twitter Developer Account
  2. Create an App
  3. Write a script (Python is a good way to go)
  4. Automate your script
